Root sandbox SDK — test UI

This page’s URL (e.g. http://127.0.0.1:3847) is only the local test server in Node. Your browser talks to that server; the server then calls Root using ROOT_BASE_URL from test-app/.env (e.g. https://api-dev01.useroot.com). Also set ROOT_API_KEY (full test_<uuid>_<secret> token). Run pnpm test-app. ROOT_SDK_DEBUG=1 logs each outbound Root request in the terminal.
